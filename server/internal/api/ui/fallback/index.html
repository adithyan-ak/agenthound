AgentHound UI not built

This server binary was compiled without the React UI. The REST API at /api/v1/* is fully functional — only the embedded UI is missing.

CLI workflows (agenthound-server ingest, query) work as-is and don't need the UI.

To build the UI and rebuild the server with it embedded, run:

make build-server

That target invokes make ui-build first, which compiles the React app and copies the output into the server binary.

See Installation guide for full instructions.
